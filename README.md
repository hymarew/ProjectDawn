# ProjectDawn テストプレイ用マニュアル

大量の敵を倒しながらWaveを攻略する3Dアクションシューティングです。
敵が落とす武器を集めて、より強い装備でステージに挑みます。

---

## 1. 起動方法

`x64\Release\ProjectDawn.exe` をダブルクリックで起動してください。
（`Data` / `asset` / `shader` フォルダはビルド時に自動コピーされるため、exe単体で動きます）

- 配布する場合は `x64\Release` フォルダごとzipにしてください
- セーブデータは exe と同じ場所の `Data\save.json` に作られます
  （**最初からやり直したい場合はこのファイルを削除**してください）

## 2. 画面の流れ

```
タイトル ─Enter→ モード選択(STORY / ENDLESS)
              ↓
        武器選択画面（装備を決める）
              ↓
   ステージ選択（STORYのみ）→ ゲーム本編 → リザルト
```

## 3. 操作方法

### メニュー画面（モード選択・武器選択・ステージ選択）

| キー | 操作 |
|---|---|
| W / S（↑ / ↓） | カーソル移動 |
| Enter | 決定・次へ進む |
| ESC | 前の画面へ戻る（武器選択のみ） |

### 武器選択画面

| キー | 操作 |
|---|---|
| W / S | 武器を選ぶ |
| **1** | 選択中の武器を **Primary** に装備 |
| **2** | 選択中の武器を **Secondary** に装備 |
| Enter | 装備を確定してゲームへ（Primary装備が必須） |

※ ステージ中は装備の変更ができません。ここで決めた2本で最後まで戦います。

### ゲーム本編

| 入力 | 操作 |
|---|---|
| W / A / S / D | 移動 |
| マウス移動 | カメラ（照準） |
| 左クリック | 攻撃（ロケットは単発、ARは押しっぱなしで連射） |
| **マウスホイール** | Primary ⇔ Secondary の武器切り替え |
| R | リロード |
| Space | ジャンプ |
| Shift | ダッシュ |
| ESC | ポーズメニュー（Resume / Retry / Title / Exit） |

### ゲームパッド（XInput対応・ゲーム本編のみ）

| ボタン | 操作 |
|---|---|
| 左スティック | 移動 |
| 右スティック | カメラ |
| A | ジャンプ |
| X | 攻撃 |
| B | リロード |
| 左スティック押し込み | ダッシュ |

※ 武器切り替えはマウスホイールのみです。

### デバッグ用キー

| キー | 操作 |
|---|---|
| 1 | デバッグUIの表示切替（武器・パーティクル等の確認用） |
| 2 | カーソル解放（スクリーンショット用） |

## 4. 武器収集のルール（ここをテストしてほしい！）

- 敵を倒すと一定確率でアイテムがドロップします
  - **結晶** = 武器（レアリティ色に光る：白=Common / 青=Rare / 紫=Epic / 橙=Legendary）
  - **箱** = 回復（触れると即回復）
- ドロップは**時間経過では消えません**。取得するか、ステージが終わるまで残ります
- 武器に触れると **仮取得** になります（画面左のトースト表示・左下の「Found: N」が増える）
- 仮取得した武器は…
  - **ステージクリア** → 正式に獲得！ 次回から武器選択画面に並びます
  - **ゲームオーバー・リタイア・タイトルへ戻る** → **失われます**
- 同じ武器は2本持てません（所持済みの武器を拾うと「ALREADY OWNED」と表示されます）

### トースト表示の意味

| 表示 | 意味 |
|---|---|
| NEW WEAPON! (ON CLEAR) | 仮取得した（クリアすれば貰える） |
| ALREADY OWNED | すでに所持している武器 |
| ALREADY FOUND | このステージ内ですでに拾った武器 |
| RECOVERED | 回復した |

## 5. テストで確認してほしいポイント

- [ ] 武器選択画面で装備を変えて、ゲーム内に反映されているか（右下の武器名）
- [ ] 武器を拾ってクリア → 再起動しても武器選択画面に残っているか
- [ ] 武器を拾ってゲームオーバー → その武器が**増えていない**か
- [ ] ドロップが時間経過で消えずに残り続けるか
- [ ] ホイールでの武器切り替え・リロード・回復の使用感
- [ ] レアリティによる性能差の体感（Legendaryは強すぎ？ Commonは弱すぎ？）
- [ ] 処理落ち・カクつきが起きた状況（敵の数・ドロップの数など）

不具合や気になった点は「どの画面で・何をしたら・何が起きたか」をメモしてもらえると助かります。
